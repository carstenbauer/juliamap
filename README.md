# Julia for Teaching: A World Map
World map visualization of places using Julia for teaching listed on https://julialang.org/teaching/ as of March 2018.

Click on the picture below to get to the [interactive map](https://crstnbr.github.io/juliamap).

A version of this map might end up on [julialang.org](http://www.julialang.org), see [this GitHub issue](https://github.com/JuliaLang/julialang.github.com/issues/764).
